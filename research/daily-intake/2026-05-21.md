## Candidate 2: Snyk ToxicSkills — ClawHub Security Audit

- **Source URL:** https://snyk.io/blog/toxicskills-malicious-ai-agent-skills-clawhub/
- **Source type:** Security research report
- **Source quality:** High
- **Evidence tier:** E4 (comprehensive audit by reputable security firm, published findings)
- **Suggested family:** security-governance
- **Is this:** new use case (security audit workflow for ClawHub skills)
- **Decision:** add to backlog
- **Notes:** Snyk audited ClawHub skills and found:
  - 534 skills with critical security issues
  - 1,467 malicious payloads detected
  - 36% contained prompt injection
  - 10.9% had secrets/credentials exposed
  - 100% of confirmed malicious skills contained malicious code
  - Attack techniques: external malware distribution, obfuscated data exfiltration, security disablement
  - Defense recommendations: treat skills like executable dependencies, inspect before install, use isolated environments
  - **New use case pattern:** "Agent Skill Security Audit" — automated vetting of ClawHub skills before installation. Not yet in atlas. Fits security-governance family.
  - Related: UseAI-pro/openclaw-skills-security (GitHub) provides curated security-first skills for detecting prompt injection, supply chain attacks, credential leaks — companion tool
