# Use Case Taxonomy

## Categories

### 1. Personal Command Center
Multi-channel personal OS: inbox, calendar, tasks, logistics, agent orchestration.

### 2. Coding & DevOps
PR review, CI/CD, bug fixing, codebase Q&A, release management.

### 3. RAG & Knowledge
Memory vault, document processing, semantic search, personal knowledge base.

### 4. Home Lab & Infrastructure
NAS monitoring, Docker management, smart home, infrastructure watchdog.

### 5. Business Automation
Email triage, customer support, lead scoring, helpdesk, intake workflows.

### 6. E-Commerce
Shopify/WooCommerce operations, order management, margin tracking, returns.

### 7. Security & Governance
Approval workflows, API-key hardening, skill vetting, audit trails.

### 8. Education
Language learning, project-based learning, spaced repetition.

### 9. Agent Monitoring
Quality evaluation, observability, cost tracing, tool-call analysis.
