# Contribution Quality Bar

## Definition of Done — Use Case Card

A use case card is complete when:

- [ ] **ID** — unique, follows `OC-XXX` format
- [ ] **Name** — clear, concise, specific
- [ ] **Status** — one of: confirmed, experimental, adapted, inferred, ai_idea
- [ ] **One-liner** — one sentence, explains the value
- [ ] **Best for** — target audience identified
- [ ] **Workflow** — step-by-step process described
- [ ] **MVP** — minimal viable implementation defined
- [ ] **Scores** — all 6 dimensions scored 1-5, time_to_value in hours
- [ ] **Evidence Tier** — E0-E5 assigned correctly
- [ ] **Safety Level** — L0-L4 assigned correctly
- [ ] **Risks** — at least 2 specific risks listed
- [ ] **Required safeguards** — at least 2 safeguards listed
- [ ] **Source quality** — high/medium/low
- [ ] **Sources** — at least 1 source URL
- [ ] **Related use cases** — cross-references to related cards
- [ ] **MVP Path** — v0 through v5 defined
- [ ] **Changelog** — creation date logged

## Definition of Done — Playbook

- [ ] **Overview** — what this playbook achieves
- [ ] **Prerequisites** — what's needed before starting
- [ ] **Setup steps** — numbered, specific, copy-ready commands
- [ ] **Safety checklist** — specific to the playbook
- [ ] **Common mistakes** — what NOT to do
- [ ] **Next steps** — where to go after completing

## Definition of Done — Anti-Pattern

- [ ] **What it looks like** — concrete description
- [ ] **Examples** — at least 2 specific bad examples
- [ ] **Why it's wrong** — clear explanation of risk
- [ ] **The fix** — what to do instead
- [ ] **Rule of thumb** — memorable one-liner

## Definition of Done — Research Intake

- [ ] **Sources evaluated** — at least 3 sources checked
- [ ] **Candidates documented** — new candidates in backlog
- [ ] **Rejections documented** — rejected sources with reasons
- [ ] **Decisions made** — add/hold/reject for each candidate
- [ ] **No inflated evidence** — E tier matches source quality

## Quality Red Flags

Reject or flag content that:
- Claims without sources
- Inflates evidence tier
- Understates safety level
- Counts skills as use cases
- Counts feature requests as confirmed
- Treats Reddit/X/YouTube as high confidence
- Duplicates existing family without adding value
- Lacks MVP path
- Has fewer than 2 risks for L2+ content
